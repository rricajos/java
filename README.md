# Master de Java

He creado este repositorio con el objetivo de analizar en profundidad cada concepto
del lenguaje Java. Entender cómo funciona internamente para tener conocimientos sólidos
antes de abordar frameworks, patrones de diseño y arquitecturas.

## Estructura

```
java/
├── basics/              # Variables, tipos, operadores, control de flujo
├── good/                # Métodos, Strings, Arrays
├── pro/                 # OOP, Excepciones, Collections
├── geek/                # Generics, Lambdas, Streams, Concurrencia, I/O
└── README.md
```
